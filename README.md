# AI Assistant

This is a simple AI chat assistant built using Rust as the backend and [Tauri](https://tauri.app/) + [Vue 3](https://vuejs.org/) for the frontend.

![AI Assistant Screenshot](screenshot.jpg)

## Recommended IDE Setup
